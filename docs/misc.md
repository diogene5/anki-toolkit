# Miscellanea

## Menu Shortcuts

On Windows/Linux, hold Alt and press a highlighted letter to activate a menu item. macOS doesn't support this feature but allows users to assign shortcuts to specific menu items instead. More details are available at Apple's support documentation.

## Debug Console

The debug console is typically used only when specifically requested by support staff or advanced users. Refer to the add-on writing guide for additional information.

### Accessing the Debug Console

**Windows/Linux:** Press Ctrl+Shift+; (control, shift, and semicolon simultaneously)

**Mac:** Press Command+Shift+; (command, shift, and semicolon simultaneously)

Note: Some non-English keyboards may require pressing : or + instead of ;

### Using the Debug Console

1. Start Anki and open the main window
2. Use the keyboard shortcut above to open the debug console
3. Paste the requested text into the top section
4. Press Ctrl+Return (Mac: Command+Return) to execute
5. Output appears in the bottom section
6. Copy and paste results back to support as needed

**Important:** Pressing Ctrl+Shift+Return will print results rather than execute the command. Avoid holding Shift when entering commands to prevent unexpected errors.
