# Contributing

## Sharing Decks Publicly

To distribute decks widely, users should synchronize them with AnkiWeb, then use the "Share" option from the deck menu. Existing shared decks can be updated this way without losing download counts or ratings.

Important consideration: AnkiWeb expects the deck to be at the same location as before. If a deck's name changes significantly (like moving from "Korean Verbs" to "Korean::Korean Verbs"), the update may fail. Users can recover the original name by downloading and importing the deck into a new profile.

When updating shared decks, previous downloaders won't receive automatic updates. They must re-download and re-import to access new material while preserving their study progress — provided neither party has altered the note type since initial import.

## Sharing Decks Privately

For limited distribution to study groups or classes, decks can be shared outside AnkiWeb. This involves exporting a single deck via File > Export with "include scheduling information" disabled, creating an .apkg file.

Distribution methods include email, websites, shared folders, or services like Dropbox or Google Drive. Both desktop and mobile versions support direct .apkg imports, though AnkiWeb itself cannot import these files.

During import, existing cards are ignored, new ones are added, and modified cards update if the note type matches. Notably, cards that have been deleted in the new apkg file will not be deleted in the user's collection, requiring direct contact for removals.
