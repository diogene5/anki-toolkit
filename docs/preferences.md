# Preferences

## Appearance

### General

**Language**
Allows you to change your display language. Translation improvements can be contributed through their translation platform.

### User Interface

**Theme**
Dark mode transforms Anki's interface to use dark colors with white text on black backgrounds. Some card templates may require modifications for proper compatibility. Version 2.1.50+ includes automatic day/night mode switching.

**User interface size**
Adjusts the scale of interface elements for better visibility.

**Reset Window Sizes**
Restores all windows to their default positions and dimensions.

**Video driver**
Selects the graphics rendering method. Software options work on more systems but run slower.

### Distractions

Display customization options during reviews:

- Hide top and bottom bars
- Enable minimalist mode for compact interface
- Reduce motion/animations
- Toggle between native and Anki styling (Mac/Linux only)

## Review

### Scheduler

**Next day starts at**
Sets when the next day's cards appear (default: 4 AM). This prevents double-day sessions for midnight studiers.

**Learn ahead limit**
Controls early card display when current deck is empty (default: 20 minutes).

**Timebox time limit**
Divides study sessions into timed blocks with progress feedback.

### Review

**Show play buttons on cards with audio**
Displays clickable audio playback controls.

**Interrupt current audio when answering**
Stops playing audio upon card responses.

**Show remaining card count**
Toggles card count visibility at screen bottom.

**Show next review time above answer buttons**
Displays scheduling information for studied cards.

**Spacebar (or enter) also answers card**
Enables Space/Enter key card responses.

## Editing

### Editing

**Paste clipboard images as PNG**
Switches from default JPG compression to lossless PNG format with transparent background support.

**Paste without Shift strips formatting**
Reverses default paste behavior regarding bold, colors, and other formatting.

**Default deck**
Controls note type and deck associations, determining which deck/note type gets selected during card addition.

### Browsing

**Default search text**
Customizes initial browser search parameters.

**Ignore accents in search (slower)**
Enables accent-insensitive text searches.

## Syncing

### Synchronisation

**Synchronize audio and images too**
Includes media files in AnkiWeb syncing.

**Automatically sync on profile open/close**
Toggles automatic synchronization with AnkiWeb.

**Periodically sync media**
Enables periodic background media synchronization.

**On next sync, force changes in one direction**
Prompts for upload or download preference, useful for overwriting accidental modifications.

### AnkiWeb Account

**Log Out** button logs out of AnkiWeb.
