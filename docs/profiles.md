# Profiles

## Profiles

Multiple users can utilize Anki on a single computer by establishing distinct profiles. Each profile maintains its own collection and settings, though add-ons are shared. Access profile configuration through File > "Switch Profile".

**Critical limitation:** "Only a single profile can be synced to an AnkiWeb account." Each user requires their own AnkiWeb account; attempting to link multiple profiles to one account will result in data loss.

The documentation notes that profiles serve users on shared computers rather than individual content separation. Those with multiple personal profiles should consolidate them by exporting decks from one profile and importing to another.

AnkiDroid does not support profiles.

## Profiles Window

The Profiles window (File > Switch Profile) enables users to:

- Open, add, rename, or delete user profiles
- Exit the program
- Restore automatic backups
- Downgrade collections for compatibility with earlier Anki versions (necessary to avoid errors when opening in older releases)
