# Packages Distributed by Linux Distributions

Linux distributions frequently customize Anki packages, which can introduce issues:

1. **Library Compatibility Problems** - Linux distributions frequently substitute different versions of dependencies like Qt without adequate testing for compatibility impacts.

2. **Outdated Versions** - Distributions may provide years-old versions of Anki or pre-release alpha/beta builds unsuitable for stable use. They typically disable built-in update notifications.

## Recommended Solution

Compiled builds of Anki are available on https://apps.ankiweb.net with most necessary libraries included and tested for compatibility.

## User Options

Users may continue with their distribution's version if preferred. However, when issues arise, reporting should go to the distribution's package maintainers rather than Anki's developers, since the official builds represent the tested and supported versions of the software.
