# Copy and Paste Issues

If you are experiencing problems with copying and pasting, please check if you are running other programs on your computer that monitor the clipboard, such as dictionary programs, clipboard managers or clipping tools.

The underlying toolkit that powers Anki can experience compatibility issues when third-party applications access the clipboard simultaneously.

Users experiencing copy-paste difficulties should identify and potentially disable any background programs that interact with clipboard functionality, as these monitoring tools may interfere with Anki's clipboard operations.
