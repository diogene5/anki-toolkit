# Installing & Upgrading Anki on Windows

## Requirements

Recent Anki releases require a computer running the 64 bit version of Windows 10 or 11.

- Anki 2.1.49 was the final release supporting Windows 7 and 8.1
- Anki 2.1.35-alternate represented the last version for 32-bit Windows systems

Users with legacy hardware can access previous versions through the official releases page on GitHub.

## Installing

1. Download Anki from https://apps.ankiweb.net
2. Save the installer file to an easily accessible location
3. Double-click the installer to run it (consult the [installation issues](installation-issues.md) page if errors occur)
4. Launch the application using the desktop shortcut after successful installation

## Upgrading

If upgrading from Anki 2.1.6+, there is no need to uninstall the previous version first. Simply close the application and follow the standard installation procedures; your card data persists through the upgrade.

For versions predating 2.1.6, or when switching between standard and alternate editions, uninstalling the old version beforehand is recommended to prevent conflicts, though this action preserves your card data separately.

Users considering downgrading should review version history documentation before proceeding.

## Add-on Compatibility

Newer Anki releases may introduce compatibility issues with certain add-ons. Users experiencing problems after upgrading can revert to previous versions available on GitHub's releases page.
