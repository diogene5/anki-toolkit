# Anki Manual - Introduction

## Mobile Clients

This manual covers the desktop version of Anki. Separate documentation exists for mobile platforms:

- [AnkiDroid Manual](https://docs.ankidroid.org/) for Android devices
- [AnkiMobile Manual](https://docs.ankimobile.net/) for iPhone and iPad

## Quickstart

Users seeking a rapid overview should visit the [Getting Started](getting-started.html) section.

## Getting Help

Support resources are available in the [Getting Help](./getting-help.html) page.

## Translations

Community volunteers have created translations of this manual in multiple languages, though versions may not reflect the latest updates:

- Bahasa Indonesia, Deutsch, Español, Français, Italiano, Polski, Português Brasileiro, русский язык, Українська, العربية, فارسى, 日本語, and 简体中文

Those interested in contributing translations to additional languages should consult the [translation docs](https://translating.ankiweb.net/anki/manual.html).

## Legacy Documentation

For users on earlier Anki versions, archived manuals are available at the [Internet Archive](https://web.archive.org/web/20240829022941/https://docs.ankiweb.net/). Information regarding older scheduler versions can be found in the FAQ on scheduler functionality.
